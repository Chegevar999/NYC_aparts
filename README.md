# Визуализация и анализ рынка недвижимости

## Описание проекта
Проект выполнен по дисциплине «Визуализация данных». Тема проекта — анализ объектов недвижимости на основе характеристик жилых домов из Ames Housing test dataset.

Так как в файле `test.csv` отсутствует целевая переменная `SalePrice`, задача машинного обучения реализована как кластеризация объектов недвижимости методом KMeans.

## Цель проекта
Выполнить первичный анализ данных, очистку, математико-статистический анализ, построить визуализации, создать интерактивный дашборд и реализовать ML-модель для группировки объектов недвижимости.

## Датасет
- Название датасета: Ames Housing test dataset
- Источник: Kaggle House Prices / Ames Housing
- Количество строк: 1459
- Количество признаков: 80
- Основные признаки: `GrLivArea`, `LotArea`, `OverallQual`, `YearBuilt`, `GarageArea`, `GarageCars`, `Neighborhood`, `TotalBsmtSF`

## Исследовательские вопросы
1. Как распределяется жилая площадь объектов недвижимости?
2. Какие районы отличаются по средней площади домов?
3. Есть ли связь между площадью гаража и жилой площадью?
4. Как уровень качества дома связан с площадью и другими характеристиками?
5. Можно ли разделить объекты недвижимости на группы по похожим характеристикам?

## ML-модель
- Тип задачи: кластеризация
- Модель: KMeans
- Признаки: `LotArea`, `OverallQual`, `OverallCond`, `YearBuilt`, `GrLivArea`, `FullBath`, `GarageCars`, `GarageArea`, `TotalBsmtSF`, `TotRmsAbvGrd`
- Метрики: Inertia, Silhouette Score
- Визуализация: PCA 2D scatter plot, elbow method
- ML-задача реализована как регрессионная модель прогнозирования цены недвижимости.
